条件和消耗逻辑说明：
A. 条件和消耗因子格式：
	|说明|条件和消耗的逻辑ID|参数1|参数2|
B. 一个技能最多有3个条件和消耗因子；
C. 逻辑ID:
	-1：非法ID； 									参数1：无; 					参数2：无
	0: 条件-个体必须有制定buff集合中的buff; 		参数1：buff 集合ID; 		参数2：无
	1: 条件-个体的生命必需小于指定的百分率； 		参数1：生命临界值; 			参数2：无
	2: 条件-目标对象必须有指定buff集合中的buff; 	参数1：buff 集合ID; 		参数2：无
	3: 条件-限制目标对象的级别； 					参数1：级别限制; 			参数2：无  （注：友好目标级别>级别限制 和 敌对目标级别<级别限制 时目标合法）
	4：消耗-魔法需求和消耗，按数值计算； 			参数1：魔法需求值; 			参数2：无
	5：消耗-魔法需求和消耗，按百分比计算； 			参数1：魔法需求百分比; 		参数2：无
	6：消耗-怒气需求和消耗，按百分比计算； 			参数1：怒气需求百分比; 		参数2：是否消耗所有怒气，1：消耗所有怒气
	7：消耗-连击点段数需求和消耗，按数值计算； 		参数1：连击点段数需求; 		参数2：无
	8：消耗-消耗所有的怒气，按数值计算； 			参数1：无; 					参数2：无
	9：消耗-取消特定的buff. 						参数1：buff 集合ID; ; 		参数2：无
	10: 条件-目标是否是自己的配偶					参数1: 无；					参数2: 无
	11: 消耗-百分比消耗HP；							参数1：HP需求百分比；		参数2：无
	12: 条件-HP大于指定百分比；						参数1：HP要求百分比；		参数2：无
